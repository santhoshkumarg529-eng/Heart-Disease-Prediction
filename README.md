# ❤️ Heart Disease Prediction

## 📌 Project Overview

This project predicts whether a patient has heart disease based on medical attributes using Machine Learning techniques. The project includes data preprocessing, exploratory data analysis (EDA), visualization, and predictive modeling.

---

## 🎯 Objective

To build a machine learning model that can accurately predict the presence of heart disease using patient health information.

---

## 📂 Dataset Features

- Age
- Sex
- ChestPainType
- RestingBP
- Cholesterol
- FastingBS
- RestingECG
- MaxHR
- ExerciseAngina
- Oldpeak
- ST_Slope
- HeartDisease (Target Variable)

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## 📊 Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Preprocessing
4. Exploratory Data Analysis (EDA)
5. Feature Engineering
6. Model Training
7. Model Evaluation
8. Prediction

---

## 📈 Machine Learning

The project uses machine learning techniques to predict whether a patient is likely to have heart disease based on the available medical data.
